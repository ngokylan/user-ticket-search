Basic
```js
import Col from '../Col';
import Row from '../Row';

<Container>
  <Row>
    <Col className="demo-col">A</Col>
    <Col className="demo-col">B</Col>
    <Col className="demo-col">C</Col>
  </Row>
</Container>
```

Fluid
```js
import Col from '../Col';
import Row from '../Row';

<Container isFluid>
  <Row>
    <Col className="demo-col">A</Col>
    <Col className="demo-col">B</Col>
    <Col className="demo-col">C</Col>
  </Row>
</Container>
```

* <i>The custom class demo-col is defined and used for the Styleguidist only.</i>
